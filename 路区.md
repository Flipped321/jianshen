## 需求概述及目标

> 概述需求核心目标，核心改动范围，需求实现目标，以及技术目标（如性能、效率、重构、体验等PRD要求之外的目标）

* 背景：
  * 目前因为线路不稳定、司机出勤不稳定，司机对线路熟悉度较低（司机门店熟悉度40%左右），配送效率和车辆装载无法持续提升。业务侧计划通过固定线路、固定出勤司机的方式持续提升司机对线路的熟悉度和配送效率，期望通过配送效率的提升来达成提升车辆装载和配送时效的目标。
* 目标：
  * 支持调度自主创建路区并绑定司机，支持多版本路区来适配不同的排线场景，系统提供规划路区的能力来辅助调度。功能上线后期望通过实现线路和司机固定，来持续提升配送效率、车辆装载和配送时效。

### 相关名词解释

1. 路区

   一个路区是地图上的**一片或多片区域**，通常包含固定的商家，包含一条排线线路。一个路区通常配置1个主配司机和1-2个备选司机，该区域内的订单主要由这些司机配送。

2. 路区模板

   一个路区模板由多个路区组成，一套模板经过算法计算可以生成一条排线结果。

#### 图层问题

1. 缩放与拖拽时的处理

   1. 配区图层、热力图层、路区图层

      该图层使用地图提供的api绘制，无需单独处理

   2. 司机图层

      司机图层属于自定义图层，需要将该图层嵌入到地图的覆盖层中，并实现实例化后图层的draw()方法

      ```
      function CustomOverlay( { canvas,  vm, driverLayer }) {
          this.canvas = canvas;
          this.driverLayer = driverLayer
          this.driversList = []
          this.vm = vm
      }
      CustomOverlay.prototype = new qq.maps.Overlay();
      CustomOverlay.prototype.construct = function () {
          //将dom添加到覆盖物层
          this.overlayProjection = this.getProjection()
      
          //设置panes的层级，overlayMouseTarget可接收点击事件
          if (this.canvas) {
              const panes = this.getPanes();
              panes.overlayMouseTarget.appendChild(this.canvas.div);
          }
      };
      CustomOverlay.prototype.draw = function (event, driversList) {
        const driverLayerParams = [];
        this.driversList.forEach(driver => {
            // 今日司机坐标为空
            if (!driver.latLng) {
                return;
            }
            let center = new qq.maps.LatLng(
                driver.latLng.latitude,
                driver.latLng.longitude
            );
            if (bounds && bounds.contains(center)) {
                const driverPosition = this.overlayProjection.fromLatLngToContainerPixel(center)
                let { x, y } =driverPosition
                const driverItem = this.getDriverParams({ x, y}, driver);
                driverLayerParams.push(driverItem);
            }
        })
        const driverLayer = new DriverLayer(
            {
                startPoint: { x: 0, y: 0 },
                shapeParam: driverLayerParams,
                vm: this.vm,
                eventTypes: EVENT_TYPES,
            },
            this.canvas
        );
        this.canvas.clear()
        this.canvas.add(driverLayer);
      };
      CustomOverlay.prototype.destroy = function () {
          this.canvas.remove()
      };
      
      const overlay = new CustomOverlay({ 
          canvas: this.canvas, 
          vm: this.vm, 
          driverLayer: this
      });
      overlay.setMap(this.vm.map);
      ```

2. 层级顺序

   地图上存在多个图层叠加时，通过zIndex属性使级别较高的图层在上层显示，具体层级顺序为：

   配区图层 < 热力图图层 < 路区图层 < 路区名称图层 < 司机图层
